# Snowflake ID Module for Roblox
Module to create realtively short and unique ID's for your proejcts in Roblox!

### Usage
| Function    | Params    | Description         |
| ---         | ---       | ---                 |
| `generate`  | *None*    | Generates random ID |

### Types
| Type           | Properties        |
| ---            | ---               |
| `SnowflakeID`  | ID: string        |
|                | Timestamp: number |
